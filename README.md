## Functions
```javascript
// Function Expression (Can not be Hoisted)
var america = function(){
  console.log('cold')
}

// Function Declaration (Can be Hoisted)
function india(){
  console.log('warm')
}

// Function Invocation/Call/Execution
india()
america()
```
